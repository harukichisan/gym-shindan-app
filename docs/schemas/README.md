# JSON Schemas (MVP)

- common.schema.json: shared $defs for all schemas
- config.schema.json: GET /api/v1/config response
- diagnosis-score.request.schema.json: POST /api/v1/diagnoses/score request
- diagnosis-score.response.schema.json: POST /api/v1/diagnoses/score response
- share.create.request.schema.json: POST /api/v1/share request
- share.create.response.schema.json: POST /api/v1/share response
- share.get.response.schema.json: GET /api/v1/share/{share_id} response
- error.schema.json: common error response
