# ジム診断ブラウザアプリ 要件定義書（たたき台）

更新日: 2026-01-29

## 0. 更新履歴

- 2026-01-29 v0.2 初版（簡単/複雑モード、上位3タイプ提示、複雑モードに不定愁訴・痛みヒアリング、タイプ辞書、質問票）

## 1. 背景・課題

初心者が「自分に合うジムが分からない」状態で入会すると、以下が起きやすい。

- 目的とジム業態がミスマッチ（例: 指導が必要なのに放置される/混雑で使えない/環境が合わない）
- 継続できない（時間帯・導線・心理的ハードル・予約負担など）
- 痛み/不調があるのに、負担が増える選択をしてしまう（不安で通えなくなる/悪化が怖くて続かない）

これらを「短時間の質問→根拠のある上位3提案→体験で確認する観点」まで落とし込んで解消する。

## 2. 目的（提供価値）

- 初心者が、**自分に合う“ジムの種類（業態）”**を上位3つに絞れる
- 上位3それぞれについて、**合う理由/注意点/見学・体験のチェック項目**が分かる
- 痛み/不定愁訴がある場合でも、**安全配慮の観点**を明示しつつ上位3を提示できる（※医療判断はしない）
- 「次に何をすれば良いか」（体験予約、質問テンプレ、最初の4週間の行動）が具体化する

## 3. 対象ユーザー

- 居住: 日本国内
- レベル: 初心者（運動習慣ほぼなし〜半年程度）
- 状況: 「ジムに行きたいが選べない」「続くか不安」「人目や痛み/不調が気になる」

## 4. 用語定義

- ジムタイプ: 店舗名ではなく、業態/形式（例: 24hジム、パーソナルなど）
- 不定愁訴: 医療診断名ではなく、本人が感じる不調（疲れやすい、睡眠の質、肩こり、頭痛など）
- セーフティ分岐: 危険な可能性がある症状が示唆される場合に、受診/相談を促す表示（上位3提示は継続）

## 5. スコープ

### 5.1 やること（スコープ内）

- 簡単診断/複雑診断の質問フロー
- 回答からのスコアリングにより、ジムタイプ上位3を提示
- 各ジムタイプの説明、料金目安、チェックリスト、注意書きの提示
- 痛み/不調の詳細ヒアリング（複雑モード）と、結果への反映
- 免責/注意喚起（医療・安全・一般情報の範囲）

### 5.2 やらないこと（スコープ外）

- 店舗検索、地図、特定店舗の推薦
- 個別のトレーニングメニュー処方（医療・指導に準ずる行為に寄らない）
- 症状の診断、治療提案、薬の推奨

## 6. 成果物（アウトプット仕様）

結果画面で必ず出す。

- おすすめジムタイプ上位3（カード形式）
- 各カード:
  - 合う理由（回答に紐づく根拠を2〜4点）
  - 注意点/合わないサイン（1〜3点）
  - 体験時チェックリスト（5〜10項目）
  - 料金目安レンジ（日本の一般的な相場。地域/店舗で変動する旨を併記）
- あなたの継続ボトルネック（1〜2個）
- 最初の4週間ミニプラン（頻度・滞在時間・やることの粒度）
- 痛み/不調がある場合:
  - セーフティ注意表示（受診/相談の促し）
  - 施設に伝えるテンプレ（例文）
  - 避けたい運用例（無理な高強度、痛みを我慢、など）

## 7. 画面・ユーザーフロー（案）

### 7.1 共通

- Start: モード選択（簡単/複雑）、免責の同意
- Question: 1画面1問（戻る/進む、進捗表示）
- Result: 上位3 + 根拠 + チェックリスト + 次アクション

### 7.2 簡単診断フロー

Start -> Q(6問前後) -> Result

### 7.3 複雑診断フロー

Start -> セクション導入 -> 基本情報 -> 生活制約 -> 嗜好/環境 -> 指導/サポート -> 設備 -> 痛み/不定愁訴（分岐） -> 確認（任意） -> Result

## 8. 機能要件

### 8.1 診断モード

- 簡単診断: 3〜7問で上位3を提示
- 複雑診断: 15〜30問（痛み/不定愁訴で追加）で上位3を提示
- 途中離脱を減らす:
  - 質問は短文、用語は初心者向け
  - 選択肢には例を添える（例: 「週2回（30分〜）」）

### 8.2 結果表示

- 上位3の順位とスコア差を「言い切らない」形で表現（例: あなたに合いやすい順）
- 根拠は回答に基づくもののみを表示（ブラックボックス感の低減）
- 痛み/不調がある場合、結果上部に注意喚起を固定表示

### 8.3 保存・共有（任意だが推奨）

- ローカル保存（localStorage）で「前回結果」を復元
- 共有: URLに短いパラメータで再現（個人情報は含めない）

### 8.4 免責/注意

- 「医療行為ではない」「一般的情報」「不安が強い/症状がある場合は医療機関へ」表示
- 痛み/不調セクション開始時と結果画面に重ねて表示

## 9. ジムタイプ辞書（表示文言 + チェックリスト + 料金目安）

注意: 料金は目安。地域、キャンペーン、オプション、入会金等で変動。

### 9.1 24hジム（マシン中心）

- 料金目安: 月額 6,000〜12,000円
- こんな人に合う:
  - 時間が不規則/夜・早朝に行きたい
  - まずは低コストで始めたい
  - マシン中心で黙々とやりたい
- 注意点（初心者の落とし穴）:
  - メニュー設計/フォームが不安だと迷子になりやすい
  - 混雑時間帯は待ちが発生し、ストレスで続かないことがある
- 痛み/不調がある場合の配慮:
  - 可能なら「フォームチェック」「初心者ガイダンス」の有無を重視
  - 無理な重量UPや反復を避け、痛みが出たら中断
- 体験時チェックリスト:
  - スタッフ滞在時間/初心者案内があるか
  - 混雑ピークの様子（自分が行く時間帯）
  - マシンの使い方表示、清掃、治安/明るさ
  - フリーウエイトエリアの雰囲気（初心者が入りやすいか）
  - 退会/休会条件、ロッカー/シャワーの快適さ

### 9.2 総合フィットネスクラブ（スタジオ・プール等）

- 料金目安: 月額 8,000〜18,000円
- こんな人に合う:
  - クラスで楽しく続けたい（スタジオ）
  - プール/サウナ等も含めて「習慣化」したい
  - ジム単体だと飽きやすい
- 注意点:
  - 利用時間が限られる/混雑で満足度が下がる場合
  - 使わない設備が多いとコスパが悪い
- 痛み/不調がある場合の配慮:
  - 低負荷クラスの有無、インストラクターのフォローを確認
- 体験時チェックリスト:
  - 自分の通える時間帯のプログラムがあるか
  - スタジオの難易度表記/初心者向け枠の有無
  - プール利用ルール、館内導線（混雑/待ち）
  - インストラクターの声かけ文化
  - オプション費用（タオル、ロッカー等）

### 9.3 公営ジム（都度利用）

- 料金目安: 1回 200〜700円（回数券/定期がある場合も）
- こんな人に合う:
  - まずはお試しで始めたい（固定費が不安）
  - 月に行ける回数が読めない
  - 近所で最低限の設備があればOK
- 注意点:
  - 指導/サポートが少ないことが多い
  - 設備が最新でない/混雑が読みにくい場合
- 痛み/不調がある場合の配慮:
  - 無理をしない前提で、軽い有酸素や可動域づくり中心にしやすい
- 体験時チェックリスト:
  - 初回講習の有無、利用ルール（服装/室内靴）
  - マシンの種類（自分の目的に足りるか）
  - 混雑/利用制限（時間枠、台数、待ち）
  - 更衣室、シャワー、ロッカーの使い勝手

### 9.4 パーソナルジム（マンツーマン）

- 料金目安: 1回 7,000〜15,000円（プランで大きく変動）
- こんな人に合う:
  - 何をすれば良いか分からない（メニュー/フォーム不安）
  - 最短で成果を出したい、強制力がほしい
  - 痛み/不調が心配で、負担調整しながら進めたい
- 注意点:
  - 料金が高め（通う頻度と予算のバランスが重要）
  - トレーナーの質の差が大きい（相性も含む）
- 痛み/不調がある場合の配慮:
  - 「痛みが出ない範囲での代替案」「フォーム/負荷の調整方針」を体験で確認
- 体験時チェックリスト:
  - カウンセリングで生活・既往・痛みを聞いてくれるか
  - フォーム指導が具体的か（なぜ/どう変える）
  - 無理な食事制限を強要しないか
  - 予約の取りやすさ/キャンセル規定
  - 卒業後のプラン（自走支援があるか）

### 9.5 セミパーソナル（少人数）

- 料金目安: 月額 15,000〜40,000円（回数で変動）
- こんな人に合う:
  - ある程度の指導がほしいが、パーソナルより費用は抑えたい
  - 一人だとサボるが、大人数は苦手
- 注意点:
  - 同時間帯の参加者次第でフォロー量が変わる
  - 予約枠が埋まると通えない
- 痛み/不調がある場合の配慮:
  - 痛みがある前提で、個別の代替メニューが出せるか確認
- 体験時チェックリスト:
  - 1枠あたりの定員、トレーナー1人あたりの人数
  - 個別対応の範囲（痛み/負荷調整）
  - 予約の取りやすさ（希望時間帯）
  - 雰囲気（初心者が置いていかれないか）

### 9.6 スタジオ系（ヨガ等のクラス中心）

- 料金目安: 月額 8,000〜16,000円（都度/回数券あり）
- こんな人に合う:
  - 一人で黙々より、クラスで習慣化したい
  - 体力に自信がなく、低〜中負荷から始めたい
- 注意点:
  - 目的が筋力アップ中心だと物足りない場合
  - 人気枠は予約が取りづらい
- 痛み/不調がある場合の配慮:
  - 初心者/やさしいクラス、代替ポーズ案内の有無を確認
- 体験時チェックリスト:
  - 難易度表記の分かりやすさ
  - インストラクターが代替案を出す文化があるか
  - 予約/キャンセルのしやすさ
  - 更衣/シャワーの要否（通いやすさ）

### 9.7 格闘技フィットネス（ボクササイズ等）

- 料金目安: 月額 9,000〜16,000円
- こんな人に合う:
  - 運動が「楽しくないと続かない」
  - ストレス解消を重視したい
  - 有酸素中心で汗をかきたい
- 注意点:
  - フォームが雑だと痛みにつながることがある（肩/腰/膝など）
  - 混雑時は指導が薄くなる場合
- 痛み/不調がある場合の配慮:
  - 痛みがある部位に配慮したフォーム/強度調整ができるか体験で確認
  - 強度を上げすぎない、休む判断ができる運用が望ましい
- 体験時チェックリスト:
  - 初心者向け導入（フォーム説明、段階的強度）
  - ミット/サンドバッグの指導密度
  - 強度調整の仕組み（軽めメニューの提示）
  - クラスの雰囲気（煽りが強すぎないか）

### 9.8 コンディショニングジム（動作評価・身体の使い方重視）

- 料金目安: 月額 10,000〜40,000円（セッション形態で変動）
- こんな人に合う:
  - 痛み/不調が不安で、負担を増やさずに運動習慣を作りたい
  - 姿勢/動作の癖を見てもらい、再現できる形で教わりたい
  - 「追い込む」より「整えてから鍛える」が合う
- 注意点:
  - 施設によって提供内容が幅広い（トレーナーの専門性、方針の差）
  - 目的が短期の減量だけだと遠回りに感じる場合
- 痛み/不調がある場合の配慮:
  - 受診中/治療中であれば、その範囲で進める方針か確認
  - 痛みを我慢させない運用（中断・代替案）を重視
- 体験時チェックリスト:
  - 評価（動作確認）→説明→再現性のある宿題、の流れがあるか
  - 専門用語を噛み砕いて説明してくれるか
  - 目的（痩せたい等）と手段（整える/鍛える）の合意が取れるか
  - 自宅でできる代替案が提示されるか

## 10. 推薦ロジック（仕様案）

### 10.1 基本方針

- 回答から「特徴タグ（traits）」を生成し、ジムタイプごとの適合度を加点/減点して総合点を算出する。
- 例: `needs_coaching`, `budget_low`, `prefers_group`, `schedule_irregular`, `pain_present` など。
- 上位3は必ず提示する（ただし痛み/セーフティ該当時は注意表示を強くする）。

### 10.2 スコア軸（内部）

内部的には以下4軸を持つ（結果には数値表示しない/しても任意）。

- 継続性: 時間/導線/予約/心理負担
- 成果: 目的達成に向くか（初心者の学習も含む）
- 心理安全性: 人目/客層/初心者導線
- コスパ: 予算と利用頻度見込み

### 10.3 タグ例（最小セット）

- 目的: `goal_weight_loss`, `goal_muscle`, `goal_health`, `goal_stress`
- 制約: `schedule_irregular`, `time_short`, `commute_easy`
- 予算: `budget_low`, `budget_mid`, `budget_high`
- 指導: `needs_coaching_high`, `needs_coaching_mid`, `self_directed`
- 嗜好: `prefers_solo`, `prefers_group`, `self_conscious_high`, `enjoys_variety`
- 痛み/不調: `pain_present`, `pain_high_impact`, `red_flag_maybe`

### 10.4 痛み/不調の補正（普通）

- `red_flag_maybe` が立った場合:
  - 結果画面の先頭に「運動前に医療機関へ相談」を表示（強め）
  - スコアは「安全配慮しやすいタイプ」を優先して上位に上げる
    - 優先: コンディショニングジム / パーソナル / セミパーソナル
    - 控えめ: 格闘技フィットネス（強度が上がりやすい前提で減点）
- `pain_present` のみの場合:
  - 指導/負荷調整のしやすさを加点（コンディショニング/パーソナル/セミパ）
  - 自走タイプ（24h/公営）は「初心者ガイダンス等の有無」を根拠文に反映

## 11. 質問票

### 11.1 簡単診断（例: 6問）

Q-S1 目的（単一選択）
- 痩せたい（体脂肪を落としたい）
- 筋肉をつけたい（見た目/体力）
- 健康維持（運動不足解消）
- ストレス解消（気分転換）

Q-S2 予算（単一選択）
- 〜7,000円/月
- 〜12,000円/月
- 〜20,000円/月
- それ以上でもOK

Q-S3 行ける時間（単一選択）
- 朝〜昼が多い
- 夕方〜夜が多い
- 深夜/早朝が多い
- 不規則（決まっていない）

Q-S4 人目（単一選択）
- かなり気になる
- 少し気になる
- あまり気にならない

Q-S5 指導の必要度（単一選択）
- 何をすればいいか分からない（教えてほしい）
- だいたい分かるが不安（時々見てほしい）
- 自分で進められる

Q-S6 好み（単一選択）
- 黙々とやりたい
- クラス/みんなでやる方が続く

### 11.2 複雑診断（分岐付き・画面化できる粒度）

記法:
- 形式: 単一選択 / 複数選択 / スライダー / 任意入力
- 分岐: `if ... then ...`

#### セクションA: 基本（目的・経験）

Q-A1 いちばんの目的は？（単一選択）
- 体脂肪を落としたい
- 体力をつけたい（疲れにくくしたい）
- 筋肉をつけたい（見た目/姿勢）
- 肩こり/腰の違和感などを減らしたい（※医療ではなく運動習慣として）
- ストレスを減らしたい/気分転換したい

Q-A2 運動経験（単一選択）
- ほぼない（ここ数年していない）
- たまに（散歩や気が向いたとき）
- 過去にジム経験あり（今は続いていない）
- いま何か続けている（週1以上）

Q-A3 ジムでの不安（複数選択）
- 何をすればいいか分からない
- フォームが不安/ケガしそう
- 人目が気になる
- 続けられる自信がない
- 予約やルールが面倒そう
- その他（任意入力）

#### セクションB: 生活制約（通いやすさ）

Q-B1 週に通えそうな回数（単一選択）
- 週1
- 週2
- 週3以上
- まだ分からない（不定）

Q-B2 1回あたりの滞在時間（単一選択）
- 30分以内
- 45〜60分
- 60〜90分
- 90分以上

Q-B3 通える時間帯（複数選択）
- 朝
- 昼
- 夕方
- 夜
- 深夜/早朝
- 不定（毎週変わる）

Q-B4 通いやすさ（単一選択）
- 自宅/職場から近い場所が必須
- 近い方が良いが、多少遠くてもOK
- 場所より内容重視

Q-B5 支払いスタイルの好み（単一選択）
- 都度払い/回数券が安心
- 月額で通い放題がいい
- どちらでも

#### セクションC: 環境・心理（続けやすさ）

Q-C1 人目（単一選択）
- かなり気になる（初心者だと分かるのが嫌）
- 少し気になる
- 気にならない

Q-C2 雰囲気（単一選択）
- 静かに黙々
- 程よく活気
- ワイワイ（声かけ多め）

Q-C3 混雑への耐性（単一選択）
- 混んでると行かなくなる
- 少しならOK
- 混雑してても大丈夫

Q-C4 予約の手間（単一選択）
- 予約が面倒だと続かない
- 予約はできるが、取りづらいのは嫌
- 予約がある方が続く（強制力がほしい）

Q-C5 グループ/個人（単一選択）
- 一人でやりたい
- 少人数ならOK
- グループが楽しい

#### セクションD: 指導・サポート

Q-D1 指導の必要度（単一選択）
- かなり必要（何をするか決めてほしい）
- ある程度必要（フォームは見てほしい）
- 基本は自分でできる

### 12.1 痛み/不調がある場合の注意表示（例）

## 13. 非機能要件（案）

- 対応環境: スマホ優先（iOS Safari / Android Chrome）、PC主要ブラウザ
- パフォーマンス: 質問画面は即時表示（体感ストレスをなくす）
- アクセシビリティ: タップ領域、文字サイズ、色コントラスト、キーボード操作（可能なら）
- プライバシー:
  - 個人が特定される情報（氏名/住所/電話等）は収集しない
  - 保存/共有は回答のみ（匿名）を前提にする
- 表現: 医療的断定を避け、一般情報としての注意喚起に留める
